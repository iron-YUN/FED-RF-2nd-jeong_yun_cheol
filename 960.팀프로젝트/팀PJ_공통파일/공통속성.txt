[ 공통 속성 도출을 위한 속성명 정의 ]

1. ui ux 구현을 위한 속성명
(0) CSS 값 정의하기 (변수화하기)
    ((전역정의))
    :root{
        --변수명1 : 값;
        --변수명2 : 값;
        --변수명3 : 값;
    }
    ((지역정의))
    요소명/클래스명{
        --변수명 : 값
    }

(1) 박스속성
   기본 5가지 : 
   가로크기 / 세로크기 /  보더 / 마진 / 패딩
   width-size height: border: margin: padding: 

(2) 글자속성 
    글자크기/글자체/글자스타일/글자두께
    글자박스 정렬 (줄간격 / 글자정렬)
     font-size: font-family: font-style: font-weight: line-height: text-align: 

(3) 미디어속성 : video태그 / img태그 / audio태그
(4) 영역정의 : 상단영역 / 메인영역 / 하단영역
(5) 요소속성 및 영역에 따른 작명하기
    -> 공통 속성 정의
    예) 
    상단영역 로고 가로크기
    --header-area-logo-width : 300px
    이름을 줄이기 위한 방법은
    상단영역의 변수로 별도설정함
    .header-area{
        --logo-width : 300px;
    }

2. JS기능 구현을 위한 속성/메서드 정의
-> 개별 개발요구사항에따라 정의도미!  

->큰영역을 정의하고 그 하위 요소를 정의함
-> 그 요소에서 필요로 하는 속성을 항목화 한다!
