[게시판 모듈구현 진행순서]

1.게시판 데이터를 위한 데이터 스키마 작성
/************************************ 
    [ 게시판 기본 데이터 구조 ]
    1. 기본키(유일키) : idx
    2. 제목 : tit
    3. 내용 : cont
    4. 첨부파일 : att
    5. 입력날짜 : date
    6. 작성자아이디 : uid
    7. 작성자이름 : unm
    8. 조회수 : cnt
************************************/

2. 임시 데이터를 생성하여 제이슨 파일로 출력
-> board.json 
(제이슨 파일을 쓴 이유는?)
-> 데이터베이스에서 데이터를 가져올 때 보통사용되는 프론트앤드에 사용할수있는 
데이터 포멧중 일반적인 것이 제이슨 파일이다!
